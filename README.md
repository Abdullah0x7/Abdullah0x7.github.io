# Abdullah Bello's Personal Website

This repository contains the source code for my personal website and blog, hosted at [abdullahbello.name.ng](https://abdullahbello.name.ng).

It is built using [Jekyll](https://jekyllrb.com/) and the [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) theme, customized to share knowledge and tutorials.

## üõ†Ô∏è Local Development

To run this site locally on your machine, follow these steps:

### Prerequisites

- [Ruby](https://www.ruby-lang.org/en/downloads/) (v2.7 or higher)
- [Bundler](https://bundler.io/)

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Abdullah0x7/Abdullah0x7.github.io.git
    cd Abdullah0x7.github.io
    ```

2.  **Install dependencies:**
    ```bash
    bundle install
    ```

### Running the Server

Start the local Jekyll server with live reload:

```bash
bundle exec jekyll serve
```

The site will be available at `http://127.0.0.1:4000/`.
